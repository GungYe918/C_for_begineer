#if 문과 마찬가지로 특정조건이 충족될 경우 {}안의 내용을 출력하는 조건문이다. 그러나 #if 문과의 
차이점은 명백하다. #if 문은 조건을 충족한 뒤 {}안의 내용을 실행하면 괄호 밖으로 빠져나오지만
#while 문은 조건이 충족되는 한 계속 반복으로 {}안의 내용을 실행한다.

#while **문의 구성은 다음과 같다.**

```
while(조건문)
{
	실행문
}
```

**다음은 while문을 실행하는 예시코드이다.**


```C
#include <stdio.h>
int main(void)
{
	int i;
	while(i<100)
	{
		printf("The Number is %d", i);
		i++;
	}
	return 0;
}
```
